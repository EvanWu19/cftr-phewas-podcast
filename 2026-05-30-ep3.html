Blurb

Sun Q, Wen J, Knowles MR, Strug LJ, Yun Li and colleagues (HGG Adv, 2022) document the imputation infrastructure of the Cystic Fibrosis Genome Project (CFGP). They TOPMed-Freeze-8-imputed 7,970 array-genotyped CF samples and validated quality using 5,095 deep (~30×) WGS samples as ground truth. TOPMed achieved average true-R² > 0.93 across MAF categories and recovered 11.2-43.1M well-imputed variants per array; F508del true-R² = 0.83-0.87. They then constructed a CF-specific reference panel from CFGP WGS — small (~10K haplotypes, 3% of TOPMed's size) but built from CF haplotypes — which outperformed TOPMed at 138 of 544 CFTR-region variants including 11 highly damaging (CADD>20) ones, and raised F508del true-R² to 0.93. The published CFGP reference panel is accessible by CFF data request. Headline gap-marker: of 281 CFTR2 disease-causing variants mappable to hg38, WGS directly covers 137, TOPMed-imputation rescues only 35 well, older 1000-Genomes-based imputation rescues 15. Proposed direction for CFTR PheWAS: build a hybrid array-only-plus-WGS unified CFF call set using the CFGP reference panel to roughly double the effective modifier-scan cohort from ~5,000 to ~9,000. Open: registry-linkage depth of the 4,656 array-only samples; data-use agreement coverage; Rsq comparability across reference panels.

Plain-English glossary

  • CADD phred score — Combined Annotation Dependent Depletion deleteriousness score on a phred scale; >20 = top 1% most damaging variants.
  • CFFPR — Cystic Fibrosis Foundation Patient Registry.
  • CFGP — Cystic Fibrosis Genome Project; CFF-sponsored consortium that produced the deep WGS on 5,095 CF samples.
  • CFTR2 — public curation database of CF-disease-causing CFTR variants.
  • Deep WGS — whole-genome sequencing at ~30× average coverage per position.
  • Eagle — population-phasing tool used inside the TOPMed Imputation Server.
  • EMMAX / EPACTS v3.3.0 — efficient mixed-model GWAS implementation; accounts for relatedness via a kinship matrix.
  • F508del — most common CFTR mutation; c.1521_1523delCTT, p.Phe508del; allele frequency 69.7% in CF patients vs 0.8-1% in general population.
  • FEV1 / KNoRMA — forced expiratory volume in one second / Kulich-normalised mortality-adjusted three-year FEV1 score, a quantitative CF lung-function trait.
  • GWAS / GWAS array — genome-wide association study / a chip that pre-selects ~10⁵-10⁶ SNP positions for direct genotyping.
  • Haplotype — the allele sequence on a single inherited chromosome copy; phasing splits a person's genotype back into their two parental haplotypes.
  • Imputation — statistical filling-in of unmeasured variant positions using a reference panel of phased haplotypes.
  • Indel — short insertion or deletion variant.
  • MAC / MAF — minor allele count / minor allele frequency.
  • METAL — fixed-effect meta-analysis tool for combining GWAS summary statistics.
  • MetaMinimac2 — meta-imputation tool that combines outputs from multiple reference panels with weighted-sum imputation.
  • Minimac4 — the imputation algorithm used by the TOPMed Imputation Server.
  • PI_HAT — PLINK estimate of identity-by-descent proportion; used to flag related sample pairs.
  • PLINK 1.9 — workhorse genotype-QC, PRS, and association toolkit.
  • PRS — polygenic risk score; per-individual weighted sum of risk allele dosages.
  • Pan-ancestry / 1000 Genomes — public reference cohorts of phased haplotypes from diverse populations.
  • Phecode — curated grouping of ICD codes into one biologically meaningful disease.
  • P+T method — pruning-and-thresholding PRS recipe; clump SNPs by LD then keep those passing a p-value threshold.
  • Reference panel — library of phased haplotypes from sequenced individuals; the substrate that imputation matches against.
  • Rsq / estimated R² (vs true R²) — Rsq is the imputation software's internal quality estimate (available without ground truth); true R² is the squared Pearson correlation between imputed dosage and actually-observed WGS genotype (only computable when WGS exists).
  • SNP / variant — single-nucleotide polymorphism / any DNA position that varies between people.
  • TOPMed Freeze 8 — NHLBI Trans-Omics for Precision Medicine reference panel built from 97,256 deeply sequenced human genomes, 308M variants.
  • TOP-LD — TOPMed-derived LD reference resource for European ancestry.
  • VEP — Variant Effect Predictor; classifies variants by functional consequence (missense, stop-gain, splice, intron, etc.).
  • WGS — whole-genome sequencing.

Methods to lift

  • TOPMed Imputation Server pipeline — strand-flip, phase with Eagle, impute with Minimac4 against Freeze 8 reference (97,256 deeply sequenced genomes, 308M variants).
  • True-R² evaluation harness — compute squared Pearson correlation between imputed dosage and WGS-derived genotype on samples with both; the only honest imputation-quality metric available.
  • CFGP cohort-specific reference panel — construct CF-specific haplotype panel via Eagle phasing of in-cohort WGS; outperforms general-population panels at allele-skewed regions (CFTR, HLA, chr9 60-66Mb, chr15 19-23Mb).
  • EMMAX in EPACTS v3.3.0 — linear-mixed-model GWAS with kinship correction; standard for related cohorts like CFF Twin-Sibling Study.
  • METAL meta-analysis — combine subset GWAS results across cohort strata.
  • P+T PRS construction in PLINK 1.9 — grid-search MAF (0.1%, 0.5%, 1%, 5%) × p-value-threshold (1, 0.5, 0.1, 0.05, 0.01, 5×10⁻³, 1×10⁻³, 5×10⁻⁴, 1×10⁻⁴, 5×10⁻⁵, 1×10⁻⁵) to find best-performing combination.
  • MetaMinimac2 meta-imputation — combine outputs from multiple reference panels by weighted sum; explicitly suggested as future direction for combining TOPMed + CFGP.
  • Reference-panel-aware QC — Rsq is not directly comparable across reference panels; documented case where TOPMed Rsq 0.80 vs CFGP Rsq 0.29 reverses on true quality (TOPMed true-R² 0.50 vs CFGP true-R² 0.94).

Proposed direction (conditional)

Conditional on inspection of CFF WGS variant calls and registry phenotype linkage: build a unified hybrid CFF cohort variant call set by TOPMed-imputing all 4,656 CFF array-only samples to the ~11-43M well-imputed-variant density Sun et al. document, optionally further imputing the CFTR region with the published CFGP cohort-specific reference panel, then meta-analyzing TOPMed-imputed-only and WGS-only strata against CFFPR phecode endpoints. Use imputation-Rsq-stratified subset analyses to control for differential variant quality between strata, with explicit awareness that Rsq is not panel-comparable. Expected hybrid effective cohort size ~9,000 versus WGS-only ~5,000.

What to verify first

  • Whether the 4,656 array-only samples are CFFPR-linked at depth comparable to the WGS-only 5,095 — the paper does not address registry linkage.
  • Whether data-use agreements for the published CFGP reference panel cover CFFPR-phenotype-linked modifier-PheWAS work, or are restricted to imputation-method validation.
  • Imputation-quality decay outside the CFTR region for the non-CFTR loci a CFTR-PheWAS would actually scan (e.g., chr11 EHF-APIP, chr20q13.2 Wright-2011 region, chr5p13 SLC9A3) — true-R² evaluation needs to be repeated at the chromosomes of interest, not chr 20 only.
  • Whether the heterogeneity in genotyping arrays across the seven Illumina chips (300K through Omni5) introduces stratum-specific QC issues that would need site-batch covariates in any combined analysis.
  • Modulator-era stratification — whether KNoRMA and other continuous phenotypes derived in the WGS-only and array-only strata are from comparable enrollment windows or differ systematically in modulator-era exposure.
  • Ancestry stratification — paper restricts to >95% European; African-ancestry and Hispanic strata in CFF may need separate-stratum imputation or skipping.
  • Meta-imputation feasibility — whether MetaMinimac2 has been used in CF before, and whether a TOPMed-plus-CFGP combined imputation gives an additional power gain over either alone in practice.

Gap-map: closes / leaves open

Closes:

  • TOPMed Freeze 8 imputation quality on CF cohorts is documented and adequate (true-R² >0.93 across MAF categories).
  • A published CF-specific imputation reference panel exists (CFGP, ~10K haplotypes from 5,095 WGS samples) and is by-application-accessible via the CFF.
  • Imputation-aided PRS for KNoRMA approaches direct-WGS performance (TOPMed: 0.99% below WGS in target-imputation scenario; 71.2% above array-only in discovery-imputation scenario).
  • The exact composition of the CFF WGS cohort by genotyping platform and source study is documented (Table S1 in the supplement).

Leaves open:

  • Registry-linkage status of the array-only subset.
  • True-R² outside CFTR for any specific CFTR-PheWAS scan target.
  • The remaining ~136 of 281 CFTR2 disease-causing variants that neither TOPMed-imputation nor older 1000-Genomes-imputation rescue well — these still require either direct WGS or targeted Sanger.
  • Meta-imputation combining TOPMed + CFGP is mentioned as a "future direction" but is not tested in the paper.
  • Non-European-ancestry imputation quality for CFF cohort subjects.
  • Modulator-era phenotype comparability across hybrid-cohort strata.

Read full transcript (20 turns · ~2069 words)

ALEX Episode three. Today's CFTR-cohort paper is unusually load-bearing for the project — Sun, Wen, Knowles, Strug, Yun Li and colleagues, 2022, in Human Genetics and Genomics Advances. Title: "Leveraging TOPMed imputation server and constructing a cohort-specific imputation reference panel to enhance genotype imputation among cystic fibrosis patients." This is the paper that documents the exact whole-genome-sequencing infrastructure on the CF cohort Runyu wants to work in.

JORDAN Slow down. This is heavy methodology. Define everything first: WGS, imputation, reference panel, TOPMed, F508del, MAF, Rsq, PRS, KNoRMA. Treat me as if I've never opened a genotype file.

ALEX Yes. Whole-genome sequencing, WGS, is reading every letter of a person's DNA — about three billion bases — using a sequencer that reads short fragments and overlays them many times to reach consensus. "Deep" coverage means about thirty-fold redundancy at each position, which is the standard for variant calling. Once you have a sequenced genome you can identify single-letter changes — SNPs, "snips" — and short insertions and deletions — indels — and call them all. That gives you a full variant call set for each person. Now contrast that with a genotyping array. A genotyping array is a chip that probes only a few hundred thousand to a few million pre-selected positions in the genome. The Illumina 610-Quad chip, mentioned in this paper, probes about six hundred ten thousand SNPs. So arrays are cheap and fast but they see only one in every five thousand positions. To bridge that gap you do imputation: you take a person's chip-genotyped positions and statistically fill in the millions of positions you didn't directly measure, using a reference panel as the guide.

JORDAN And what's the reference panel?

ALEX A reference panel is a precomputed library of phased haplotypes — phased meaning each chromosome split into its two parental copies — from a population of people whose entire genome has been deeply sequenced. When you impute, the algorithm finds the closest-matching reference haplotypes to your person's chip-genotyped letters in a small genomic window, and copies over the missing letters. TOPMed is the National Heart, Lung, and Blood Institute's Trans-Omics for Precision Medicine programme. Its Freeze 8 reference panel was built from ninety-seven thousand two hundred fifty-six deeply sequenced genomes from the general U.S. population and contains three hundred eight million variants. The CF version that this paper builds is the CFGP reference panel, the Cystic Fibrosis Genome Project panel, constructed from the WGS of five thousand ninety-five CF patients — vastly smaller, but every haplotype comes from someone who actually has CF.

JORDAN F508del, MAF, Rsq?

ALEX F508del is the single most common CFTR mutation — a three-base-pair deletion at codon five hundred eight of the CFTR gene that removes a phenylalanine residue. Its scientific shorthand is c.1521 underscore 1523 del C T T, p Phe five oh eight del. In CF patients it has an allele frequency of about sixty-nine-point-seven percent — meaning roughly seven out of every ten chromosomes in the patient population carry it. In the general population, by contrast, its frequency is zero-point-eight to one percent. So this allele is dramatically enriched in CF. MAF is minor allele frequency: at any SNP, the rarer of the two alleles, expressed as a fraction. Rsq, written R-squared, is the imputation quality metric output by the imputation software — minimac4 in this paper — and it estimates how confidently the algorithm thinks it called each imputed dosage. A threshold of zero-point-eight is the strict-keep standard; zero-point-three is the lenient standard. PRS, polygenic risk score, is the per-individual sum of weighted-risk-allele dosages we covered yesterday. KNoRMA is a CF-specific lung-function summary score — a Kulich-normalised mortality-adjusted three-year FEV1 measure — that CF modifier researchers use as a continuous outcome. FEV1 is forced expiratory volume in one second, the workhorse spirometry measure of lung function.

JORDAN OK. Why was this paper needed?

ALEX Two specific gaps. First, previous CF-cohort imputation used the Haplotype Reference Consortium panel from 2016, with around sixty-four thousand four hundred haplotypes from the general population. TOPMed Freeze 8 is much bigger and was hypothesised to do better — particularly for rare variants. Second, the F508del problem. A general-population reference panel barely sees F508del — it's at one percent — so when you ask it to impute F508del in a CF patient where the allele is at seventy percent, the algorithm's haplotype-matching breaks down. It systematically calls homozygous deletion as heterozygous. The team wanted to know whether building a CF-specific panel from five thousand-plus CF WGS samples — even though that's only three percent of TOPMed's sample size — could outperform TOPMed at CFTR and at other allele-skewed regions, because the haplotype context would match.

JORDAN How do they tackle it?

ALEX The cohort itself first. Seven thousand nine hundred eighty-eight CF samples were genotyped across seven different Illumina arrays — three hundred K, three hundred seventy K, six hundred sixty K, six hundred ten Quad, six hundred sixty W set one, six hundred sixty W set two, and Omni Five — across multiple studies: the CF Twin and Sibling Study TSS, the Canadian Gene Modifier Study CGS, the Gene Modifier Study GMS, EPIC, FrGMC, and CF-related Diabetes and Liver Disease substudies. Five thousand ninety-five samples got deep WGS via CFGP. After QC, seven thousand nine hundred seventy array-genotyped samples were TOPMed-imputed via the TOPMed Imputation Server using eagle for phasing and minimac4 for imputation. Then they did three things. First, they used the WGS calls as gold standard to compute the "true R squared" — the actual squared Pearson correlation between imputed dosages and observed WGS genotypes — for the two arrays with the biggest sample sizes, the 610-Quad and the 660W-set-one, on chromosome twenty as a representative case. Second, they built a reduced CFGP reference panel from twenty-eight hundred fifty WGS samples and used it to impute nineteen ninety-two held-out samples, comparing genome-wide and CFTR-region quality against TOPMed. Third, they built a full CFGP reference panel from all five thousand ninety-five WGS samples — about ten thousand haplotypes — for community use, and tested PRS performance for KNoRMA under imputation versus array-only versus WGS-direct.

ALEX One design choice I want to push on: they restrict ancestry to primarily European (over ninety-five percent of CF patients are European-ancestry, defined by principal-component analysis against 1000 Genomes anchors), which is realistic for the cohort but limits what we can say about non-European CF subjects. Same caveat as Zheutlin from yesterday.

JORDAN What did they find?

ALEX TOPMed-based imputation of CF samples is high quality. They recovered eleven-point-two to forty-three-point-one million well-imputed variants per array (Rsq above zero-point-eight), of which two-point-five to thirty-three-point-four million were low-frequency or rare, MAF below zero-point-five percent. For the largest array, the 610-Quad with thirty-eight hundred forty samples, the panel yielded forty-three million well-imputed variants. Average true-R-squared across all MAF categories was above zero-point-nine-three. That's a two-to-three-fold improvement in low-frequency-variant coverage over the older HRC panel. Now F508del specifically. TOPMed-imputed estimated Rsq was zero-point-eight-nine to zero-point-nine-three across arrays, true R squared zero-point-eight-three to zero-point-eight-seven — decent but with a long tail of misclassified homozygotes-as-heterozygotes. The reduced CFGP panel, fewer samples but CF-specific haplotypes, hit a true R squared of zero-point-nine-three for F508del — about a ten-point absolute improvement. Across the five hundred forty-four CFTR variants tested, one hundred thirty-eight were better imputed by reduced CFGP than by TOPMed, including eleven highly-damaging variants with CADD phred scores above twenty — an eight-fold enrichment over the genome-wide expectation. They highlight five exemplar variants. rs77284892, p Glu60Lys, a CFTR stop-gain — CFGP true R squared zero-point-nine-nine-seven versus TOPMed zero-point-three-three-three. rs139573311, p Leu467Pro missense — CFGP zero-point-nine-seven versus TOPMed two-point-five times ten to the minus seven, essentially uncallable from TOPMed. rs76713772, a splice acceptor — CFGP zero-point-nine-eight versus TOPMed zero-point-seven-eight. Genome-wide, TOPMed beats reduced CFGP in essentially every one-megabase window except CFTR itself and a handful of others — interestingly the HLA region on chromosome six, a stretch on chromosome nine from sixty to sixty-six megabases, and a stretch on chromosome fifteen from nineteen to twenty-three megabases, all outperform TOPMed when the reduced CFGP panel is used. The HLA finding makes biological sense — HLA has unusual haplotype structure even in general populations. The chromosome nine and chromosome fifteen findings are unexplained in the paper. PRS for KNoRMA improved by thirty-seven-point-two percent with TOPMed imputation versus array alone in the target-imputation scenario, only zero-point-nine-nine percent inferior to direct WGS. In the discovery-imputation scenario, TOPMed-aided summary stats gave seventy-one-point-two percent higher PRS-KNoRMA correlation than array-only. The headline gap-marker: of two hundred eighty-one CFTR2-curated CF-disease-causing variants mappable to hg38, CFGP WGS data directly covers one hundred thirty-seven; TOPMed-based imputation rescues only thirty-five well, an old 1000-Genomes-panel rescue rescues only fifteen. So WGS is still required for the majority of disease-causing variants, but TOPMed-imputation roughly doubles the coverage you'd get from any genotyping-array-plus-imputation pipeline.

JORDAN That's a lot of methods substance. Translate this into a research direction for Runyu.

ALEX One possible direction. The Sun et al. paper makes the cohort's data architecture explicit: roughly five thousand WGS samples, plus about four thousand six hundred fifty-six samples that have array data only, plus over three thousand samples with both. The full CFGP imputation reference panel built here was published with the intent that "other investigators studying CF" use it — that includes the modifier-PheWAS framework we keep gravitating toward. So one direction Runyu could pursue is a hybrid-cohort PheWAS that leverages the published CFGP imputation reference panel to bring the array-only four-thousand-six-hundred-fifty-six samples into a unified WGS-quality call set, expanding effective cohort size from the WGS-only five thousand to roughly nine thousand for purposes of common-variant and well-imputed-rare-variant modifier scanning. Plain English: rather than restricting to the five thousand WGS-only samples for a CFFPR-linked modifier PheWAS, use the public CFGP reference panel to impute the four-thousand-six-hundred-plus array-only samples and run scans on the combined cohort with imputation-quality stratification. Formal version: build a unified hybrid CFF-cohort variant call set by TOPMed-imputing all CFF array-only samples to ~30M well-imputed variants, then meta-analyzing TOPMed-imputed-only and WGS-only strata against CFFPR phecode endpoints with imputation-Rsq-stratified subset analyses to control for differential variant-quality between strata.

JORDAN Conditional on that direction, what does this paper close, and what does it leave open?

ALEX What it closes: that TOPMed imputation works well for CF cohorts genome-wide; that CFTR-specific imputation can be substantially improved using the CFGP reference; that imputation-aided PRS for KNoRMA approaches direct-WGS performance; and that combined-cohort analysis is methodologically tractable. What it leaves open for that direction. First — gating constraint — we'd need to verify that the four-thousand-six-hundred-fifty-six array-only samples are CFFPR-linked at the same depth as the WGS samples; the paper doesn't address registry-linkage status. Second, the CFGP reference panel as published is by-application access through the Cystic Fibrosis Foundation — we'd need to confirm that data-use agreements support the proposed hybrid-cohort use and whether the CFFPR-linked subset would even be releaseable to a non-credentialed investigator. Third, the paper's true-R-squared evaluation is from chromosome twenty in samples that already have WGS. For CFFPR phenotypes that depend on rare-variant calls outside the CFTR region — say, a modifier-locus scan on chromosome eleven for EHF-APIP — we'd need to check that the array-only samples retain enough imputation quality for the actual targets, which means re-doing this evaluation at the chromosomes that matter for our hypotheses. Fourth, the F508del finding underlines a subtle issue: imputation-quality metrics like Rsq are not comparable across reference panels — the paper documents one case where TOPMed Rsq is zero-point-eight and CFGP Rsq is zero-point-two-nine but the true quality is reverse, with CFGP true R squared zero-point-nine-four versus TOPMed zero-point-five-zero. So any QC filtering at the cohort level needs to be aware that Rsq is reference-panel-relative. Fifth, the modulator-era confound from yesterday's discussion does not affect the genotype call set itself, but does affect whether KNoRMA or any continuous lung-function phenotype derived from the same era window is comparable across the WGS-only versus array-only strata if the strata differ in modulator-era enrollment.

JORDAN That's a serious set of pre-checks before committing. What's tomorrow?

ALEX Tomorrow we close out the Sun-2022-era methods family with another CFTR-modifier paper from the same UNC group. The one-line takeaway today: the CFF WGS cohort already has a published infrastructure paper, and the cohort-specific imputation reference panel it ships can roughly double the available CF-modifier-scan sample size — but only with careful imputation-quality stratification.
