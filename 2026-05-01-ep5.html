Blurb

The AstraZeneca team's parallel pipeline to Backman 2021. Same biobank, same era, different methodological emphasis: gene-level collapsing analysis across twelve qualifying-variant masks, with a synonymous-variant empirical-null calibration cross-checked against an n-of-1 phenotype-permutation null. Both nulls converge on a study-wide significance threshold of p less than or equal to 2 x 10^-9. All summary statistics are exposed as a free public PheWAS portal at azphewas.com, covering roughly 18,762 phenotypes (binary plus quantitative) on 281,104 sequenced UK Biobank exomes (European-ancestry primary analysis cohort 269,171). Roughly 11,000 CFTR heterozygotes are present in the cohort — a candidate first-tier external replication panel for any CFTR-anchored rare-variant signal we eventually work on.

Plain-English glossary

  • AZ-PheWAS portal (azphewas.com) — AstraZeneca's free, no-login web front-end exposing gene-by-trait summary statistics from the Wang 2021 analysis. Type in a gene, see every associated trait across every mask.
  • Bonferroni correction — divide your significance threshold by the number of tests you ran, to keep the false-positive rate under control.
  • Burden test — statistical test that aggregates rare variants in a gene into a single per-person count and tests that count against a trait. Collapsing analysis is a stricter cousin.
  • Collapsing analysis — count rare variants in a gene per person under a quality filter, then test that count against a trait. One well-powered test per gene instead of thousands of underpowered ones.
  • Dominant model — one bad copy of a gene is enough to count the person as a carrier in the test.
  • Empirical-null calibration — instead of using a theoretical p-value threshold, run the entire pipeline on something that biologically shouldn't matter (synonymous variants) and use the noise distribution you observe to set the real bar.
  • Exome — the protein-coding portion of the genome, roughly one and a half percent of all DNA, where most disease-relevant rare variants live.
  • False discovery rate (FDR) — controls the expected fraction of false positives among the things you call hits. Softer than Bonferroni.
  • Heterozygote (carrier) — person carrying one copy of a rare variant in a gene. About 11,000 CFTR heterozygotes sit in the AZ-PheWAS cohort.
  • Linear mixed model (LMM) — regression that builds in a correction for relatedness and population structure, so family resemblance doesn't fool the test.
  • Minor allele frequency (MAF) — how common a variant is in the population. Common variants are above one percent; rare variants below one percent, often below zero point one percent.
  • N-of-1 phenotype permutation — randomly scramble the trait labels across people, rerun the test, and use the resulting p-value distribution to define what "by chance" looks like for your specific pipeline.
  • PheWAS (phenome-wide association study) — scan one genetic factor against thousands of traits at once, instead of one trait against millions of variants.
  • Pleiotropy — one gene affecting multiple distinct traits.
  • Recessive model — both copies of a gene need to carry a qualifying variant for the person to count as a carrier.
  • Summary statistics — post-analysis tables (gene, trait, effect size, p-value) that can be shared publicly without sharing patient-level data.
  • Synonymous variant — DNA change in a coding region that does not change the protein, because of redundancy in the genetic code. Used as a biological placebo.
  • Variant mask — a recipe specifying which variants count for a collapsing test (rarity filter, predicted-deleteriousness filter, etc.). Wang ran twelve.

Methods to lift

  • Twelve-mask collapsing-analysis design. Stratify rare variants by rarity threshold and predicted-deleteriousness category. Ten dominant non-synonymous masks plus one recessive mask gives a defensible variant-quality grid for any future CFF rare-variant work.
  • Synonymous-variant empirical null. Run the same collapsing pipeline on synonymous variants and use the resulting p-value distribution as the real noise floor. Low-cost, biologically grounded calibration layer.
  • Dual-null cross-check. Pair the synonymous empirical null with n-of-1 phenotype permutation. Convergence between the two methods on the significance threshold is the credibility receipt.
  • Public summary-statistics portal as first-tier replication. Use AZ-PheWAS for free external lookup of any CFTR-anchored signal before negotiating any other data access.

Proposed direction (conditional)

If a CFTR-anchored phenotype-modifier signal eventually emerges from a CFF rare-variant analysis, AZ-PheWAS is the obvious first-tier external replication panel — roughly 11,000 CFTR heterozygotes are queryable through the public portal at no cost and no data-access friction. Methodologically, adopt synonymous-variant collapsing as an empirical-null calibration layer in any rare-variant pipeline we end up running, and use Wang's twelve-mask grid as a template for variant-quality stratification.

What to verify first

  • Check carrier-only subgroup sizes in AZ-PheWAS for the specific phenotypes that look most promising — 11,000 carriers sounds large, but a 2-percent-prevalent phenotype leaves only ~220 cases.
